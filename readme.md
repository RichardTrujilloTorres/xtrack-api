# xTrack API

[![Build Status](https://travis-ci.org/RichardTrujilloTorres/xtrack-api.svg?branch=master)](https://travis-ci.org/RichardTrujilloTorres/xtrack-api)

The xTrack utility RESTFul Lumen API.

## Install & Build
```
composer install 
php artisan migrate
```

## Features
- CORS support
- JWT token authentication
